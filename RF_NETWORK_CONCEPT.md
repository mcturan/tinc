# RF NETWORK — CONCEPT

**Durum:** PLAN AŞAMASI — kod yok

## Kapsam
APRS, LoRa ve mesh sistemlerini birleştiren iletişim katmanı

## Planlanan Bileşenler
- APRS entegrasyonu (pozisyon, mesaj, telemetri)
- LoRa mesh ağı (internet bağlantısı olmadan haberleşme)
- AIS entegrasyonu (deniz trafiği)
- Mesh network routing

## TINC Entegrasyonu
events_rfnetwork koleksiyonu üzerinden event üretecek
Diğer uygulamalar (QRVEE harita, PNOT) event consume edecek

## Uygulama Zamanlaması
FAZ 9+ — tüm temel uygulamalar tamamlandıktan sonra
